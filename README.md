# Infinite-ui
A lightweight jQuery UI library

##Version
`1.1.1`
##Support
* Internet Explorer 8+

* Chrome 4+

* Firefox 5+

* IOS 6+

* Android 4.4.0+

## Installation


Install using npm:

```shell
$ npm install
```


## Usage

```shell
$ gulp build				//编译全部组件及css
```

```shell
$ gulp build:pc				//编译pc组件及css
```

```shell
$ gulp build:mobi			//编译mobile组件及css
```
